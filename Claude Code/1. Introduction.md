First, you’ll need a Claude plan. The free version works, but the results won’t be as strong.
Next, install Claude Code. The best approach is to follow the official Claude documentation for setup.
After that, run `claude` in your terminal and answer the prompts you’re given.
If you’re having trouble installing Claude Code, it may be worth taking a basic programming course to build the necessary foundations.
It’s also often better to establish a solid foundation before asking the AI to build features. The model will base its output on your existing code, architecture, and patterns, so setting clear structure and principles early leads to much better results in my experience. 

## Terminal & IDE
You can use the integrated terminal in your IDE, which is usually set up by default. If it isn’t, you can link your IDE and Claude Code using the `/ide` command.
